二. 构造方法解析
1.HashMap()
此构造方法只是给HashMap中的加载因子loadFactor进行赋值为默认值 0.75f
2.HashMap(int initialCapacity)
此构造方法，以initialCapacity, 0.75f为参数调用第三个构造方法。

3.HashMap(int initialCapacity,float loadFactor)
此构造方法存在逻辑：
如果initialCapacity小于0，抛出非法参数异常
如果initialCapacity大于2^30，将其修正为2^30
如果加载因子loadFactor不大于0或者是NaN(类似1/0)，抛出非法参数异常
将实例的加载因子赋值：this.loadFactor = loadFactor
计算出下次扩容临界值并给下次扩容临界值赋值：this.threshold = tableSizeFor(initialCapacity);
计算方法：就是计算出cap的对应的二次方的值，比如当cap = 2时， 返回 2
当cap = 3 ,返回4， 返回的数肯定大于或等于cap
static final int tableSizeFor(int cap) {
int n = cap - 1;
n |= n >>> 1;
n |= n >>> 2;
n |= n >>> 4;
n |= n >>> 8;
n |= n >>> 16;
return (n < 0) ? 1 : (n >= (1<<30)) ? (1<<30) : n + 1;
}
4.HashMap(Map m)
此构造方法的逻辑是先给加载因子赋值： this.loadFactor = 0.75f
然后判断m的大小是否大于0，
如果不是，什么都不做，结束。
如果是
则判断table是否为空：
如果是： 计算出map的容量大小(m的大小/加载因子 + 1)t，然后进行检查修正。如果t>threshold , 给threshold赋值。
如果table不为空，判断m的大小是是否大于了threshold,如果大于了，则进行扩容。
然后循环m, 取出key,value进行putVal操作。

三.常用方法解析
1.resize() 扩容方法
final Node<K,V>[] resize() {
Node<K,V>[] oldTab = table;
int oldCap = (oldTab == null) ? 0 : oldTab.length;
int oldThr = threshold;
int newCap, newThr = 0;
if (oldCap > 0) {
if (oldCap >= MAXIMUM_CAPACITY) {
threshold = Integer.MAX_VALUE;
return oldTab;
}
else if ((newCap = oldCap << 1) < MAXIMUM_CAPACITY &&
oldCap >= DEFAULT_INITIAL_CAPACITY)
newThr = oldThr << 1; // double threshold
} else if (oldThr > 0) // initial capacity was placed in threshold
newCap = oldThr;
else { // zero initial threshold signifies using defaults
newCap = DEFAULT_INITIAL_CAPACITY;
newThr = (int)(DEFAULT_LOAD_FACTOR * DEFAULT_INITIAL_CAPACITY);
}
if (newThr == 0) {
float ft = (float)newCap * loadFactor;
newThr = (newCap < MAXIMUM_CAPACITY && ft < (float)MAXIMUM_CAPACITY ?
(int)ft : Integer.MAX_VALUE);
}
threshold = newThr;
Node<K,V>[] newTab = (Node<K,V>[])new Node[newCap];
table = newTab;
if (oldTab != null) {
//这里的操作是将旧的table的数据复制到新的table中
}
return newTab;
}

2.put(K key, V value)
首先，计算key的hash值，计算逻辑如下：
如果key==null,则其hash = 0, 否则
h = key.hashCode() 其中key.hashCode()得到的是一个伪随机数。
hash = h^(h>>>16)
然后将，hash, key,value,false,true作为入参调用putVal方法。

3.putVal(int hash, K key, V value,boolean OnlyIfAbsent，
boolean evict) 存储方法

final int hash;//hash码，构造时指定

V value;//值

Node(int hash, K key, V value, Node<K,V> next) {
this.hash = hash;
this.key = key;
this.value = value;
this.next = next;
