1.什么是OutputStream？（输出流）

在JavaAPI中，将可以向目的地写入一个字节序列的对象称为输出流对象。
字节序列的来源地可以是文件，也可以是网络，还可以是内存块等等。
输出流根据每次写出的字节数量的不同分为字节输出流和字符输出流。
字节输出流每次都是写出一个字节的，而字符输出流每次写出都是根据基于两字节的字符为单位写出的。
