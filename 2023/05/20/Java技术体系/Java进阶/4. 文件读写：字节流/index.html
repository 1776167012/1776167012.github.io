1.д»Җд№ҲжҳҜеӯ—иҠӮжөҒ

еӯ—иҠӮжөҒжңҖеӨ§зҡ„зү№зӮ№жҳҜпјҢжҜҸж¬Ўи°ғз”ЁreadжҲ–writeиҜ»е…ҘжҲ–еҶҷеҮәзҡ„ж•°жҚ®жҳҜд»Ҙеӯ—иҠӮдёәеҚ•дҪҚзҡ„пјҢиҖҢжңүзҡ„еӯ—з¬ҰйңҖиҰҒдёӨдёӘеӯ—иҠӮеҺ»еӯҳеӮЁпјҢ
жүҖд»Ҙеӯ—иҠӮжөҒиҜ»еҸ–ж•°жҚ®дјҡеҸ‘з”ҹиҜ»еҲ°зҡ„ж•°жҚ®дёҚеҮҶзЎ®зҡ„жғ…еҶөгҖӮ(е°ұдјҡзңӢеҲ°д№ұз Ғзҡ„ж•Ҳжһң)
еңЁJavaдёӯзҡ„еӯ—иҠӮжөҒзҡ„зұ»е‘ҪеҗҚдёҖиҲ¬жҳҜд»ҘStreamз»“е°ҫзҡ„,жҜ”еҰӮFileInputStream.

1.1 еӯ—иҠӮж•°з»„жөҒ

иҫ“е…Ҙиҫ“еҮәжөҒзҡ„жәҗ е’Ң зӣ®зҡ„ең°йҷӨдәҶеҸҜд»ҘжҳҜвҖңж–Үд»¶вҖқпјҲзЎ¬зӣҳпјүеӨ–пјҢиҝҳеҸҜд»ҘжҳҜи®Ўз®—жңәеҶ…еӯҳпјҢжҲ‘д»¬д№ҹеҸҜд»ҘдҪҝз”Ёж•°з»„жөҒжқҘе°Ҷж•°жҚ®иҫ“е…Ҙиҫ“еҮәеҲ°еҶ…еӯҳгҖӮ
еӯ—иҠӮж•°з»„жөҒе°ұжҳҜз”ЁжқҘе°ҶеҶ…еӯҳдёӯзҡ„ж•°жҚ®иҫ“е…ҘеҲ°зЁӢеәҸдёӯ,жҲ–иҖ…е°ҶзЁӢеәҸзҡ„ж•°жҚ®иҫ“еҮәеҲ°еҶ…еӯҳдёӯеҺ»зҡ„еӯ—иҠӮжөҒгҖӮ

1.1.1 ByteArrayInputStream еӯ—иҠӮж•°з»„иҫ“е…ҘжөҒ

ByteArrayInputStreamжҳҜиҫ“е…ҘжөҒInputStreamзҡ„дёҖз§Қе®һзҺ°пјҢе®ғзҡ„жәҗеҸӘиғҪжҳҜеӯ—иҠӮж•°з»„byte[]пјҢе®ғж”ҜжҢҒmarkгҖӮ

жһ„йҖ ж–№жі•

ByteArrayInputStream(byte[] buf);
жӯӨжһ„йҖ ж–№жі•зҡ„еӯ—иҠӮж•°з»„жөҒзҡ„жәҗжҳҜеҸӮж•°bufжҢҮе®ҡзҡ„ж•°з»„зҡ„е…ЁйғЁеӯ—иҠӮеҚ•е…ғгҖӮ
ByteArrayInputStream(byte[] buf ,int offset,int length);
жӯӨж–№жі•жһ„йҖ зҡ„еӯ—иҠӮж•°з»„жөҒзҡ„жәҗжҳҜbufжҢҮе®ҡзҡ„ж•°з»„д»ҺoffsetеӨ„жҢүйЎәеәҸеҸ–lengthдёӘеӯ—иҠӮеҚ•е…ғгҖӮ

еёёз”Ёж–№жі•

public int read();
иҜҘж–№жі•еҸҜд»ҘйЎәеәҸең°д»ҺжәҗдёӯиҜ»еҸ–дёҖдёӘеӯ—иҠӮпјҢиҝ”еӣһиҜ»еҮәзҡ„еӯ—иҠӮеҖјгҖӮ
public int read(byte[] b,int off,int len);
иҜҘж–№жі•еҸҜд»ҘйЎәеәҸең°д»ҺжәҗдёӯиҜ»еҮәеҸӮж•°lenжҢҮе®ҡзҡ„еӯ—иҠӮж•°пјҢе№¶е°ҶиҜ»еҮәзҡ„еӯ—иҠӮеӯҳж”ҫеңЁеҸӮж•°bжҢҮе®ҡзҡ„ж•°з»„дёӯпјҢеҸӮж•°offжҢҮе®ҡж•°з»„bиҜ»еҮәеӯ—иҠӮзҡ„иө·е§ӢдҪҚзҪ®пјҢиҝ”еӣһе®һйҷ…иҜ»еҮәзҡ„еӯ—иҠӮдёӘж•°пјҢиӢҘжңӘиҜ»еҮәеӯ—иҠӮreadж–№жі•иҝ”еӣһ-1.
