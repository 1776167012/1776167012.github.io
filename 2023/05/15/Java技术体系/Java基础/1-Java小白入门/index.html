2.了解java原理

3.第一个java程序

首先，我在D目录的一个叫JAVA程序设计里的day01文件夹里建立了一个HelloWorld.java文件，然后在cmd里输入d:回车，然后输入cd D:\JAVA程序设计\day01来到day01文件夹。然后输入javac HelloWorld.java（使用java编译器将java文件编程成class字节码文件）。之后用java虚拟机运行，输入java HelloWorld（使用java虚拟机运行class字节码文件）。得到答案

大概就是这么个过程。这只是用记事本来开发的，真正的开发一般用各大厂商发布的官方集成开发工具，它能将上述过程自动完成，所以我们只需要写代码就可以了。

4.JDK和JRE

JDK是java开发工具包，它是编写java程序的程序员使用的软件，它包含了JRE。
JRE是允许java程序的用户使用的软件,是java运行环境，它包含JVM。

5.在进行下一步学习之前，我们可以了解下java关键特性

简单性（java简单易学（相对于C++））
面向对象（Java的面向对象特性与C++旗鼓相当甚至更胜一筹）
分布式，健壮性，安全性，体系结构中立，可移植性，解释型，高性能，多线程，动态性
