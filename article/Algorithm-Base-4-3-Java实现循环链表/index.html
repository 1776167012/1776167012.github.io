1.循环链表

1.1 什么是循环链表？

顾名思义，就是可以循环的链表哈哈哈，说白了就是收尾相连的链表，
也就是说在这个链表里，任何节点都可能是头节点，也可以是尾节点。

为了简单理解，这里只演示单向循环链表。


从图中可以看出，他和单向链表基本没差别，无非就是将尾节点的next设置为头节点即可（注意这个点就理解了），
也就是说单向循环链表的定义和单向链表的定义基本一致，只是多了一个标志，用于结束循环。
