因为字节流不能很好地操作本身容量单位较大的数据（一个字符有时等于多个字节）,所以需要字符流。
如果存在那种由多个字节组成的字符，那么用字节输入流读取，则会乱码！！！
但是，如果用字符输入流读取，就不会出现乱码！！ 所以为了保险起见，对于文本文件,尽量少用字节流,多用字符流

1.文件字符IO流

文件字符IO流和文件字节IO流对应，它读入和写出的数据是以字符为单位的。

它的好处在于我们可以用它来有效率的读写文件。
